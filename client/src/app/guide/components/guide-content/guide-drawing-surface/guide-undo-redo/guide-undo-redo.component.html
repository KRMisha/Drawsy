Annuler-refaire

Description

Lorsque vous n'êtes pas certains de la dernière action effectuée, il est possible d'annuler ou de refaire cette action. Pour ce faire, vous n'avez qu'à appuyer sur les touches Ctrl + Z pour annuler une action, ou appuyer sur les touches Ctrl + Shift + Z pour refaire une action. Il est aussi possible d'effectuer ces commandes grâce aux boutons situés au bas des options de la barre latérale. Cette fonctionnalité est utile pour corriger rapidement une erreur ou pour comparer le résultat d'un nouvel ajout avec ce qu'il y avait précédemment sur le dessin.

Utilisation

  • Annuler une action avec les touches Shift + Z
  • Refaire une action avec les touches Ctrl + Shift + Z
  • Appuyer sur les boutons de commandes disponibles sur la barre latérale
