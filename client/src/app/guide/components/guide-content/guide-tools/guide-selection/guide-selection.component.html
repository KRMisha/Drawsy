Outil de sélection

Description

La sélection permet de déplacer certains éléments spécifiques de votre dessin. Si vous souhaitez déplacer un seul élement, il suffit d'un simple clic gauche sur celui-ci pour le sélectionner. Vous pouvez aussi sélectionner un groupe d'éléments en gardant le clic gauche enfoncé et en choisissant la zone à sélectionner avec un déplacement de la souris. Enfin, il est aussi possible de sélectionner le dessin en entier à l'aide des touches Ctrl + A lorsque l'outil de sélection est choisi. Une fois la zone sélectionnée, un clic gauche maintenu sur la zone et un déplacement de la souris permet le bouger tous les éléments à la position souhaitée.

Utilisation

  • Sélectionner l’icône de sélection dans la barre d’outils
  • Effectuer un clic gauche sur un élément pour le sélectionner
  • Effectuer un clic gauche maintenu pour sélectionner une zone
  • Appuyer sur Ctrl + A pour sélectionner tout le dessin
  • Effectuer un clic gauche maintenu sur la zone pour déplacer les éléments
