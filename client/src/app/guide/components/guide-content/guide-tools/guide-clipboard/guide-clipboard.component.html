Presse-papier

Description

Le presse-papier permet de créer des dessins d'un tout autre niveau. Il permet de copier une sélection avec les touches Ctrl + C, et ensuite de coller une sélection à l'aide de Ctrl + V. La duplication, effectuée avec les touches Ctrl + D, permet essentiellement de copier et de coller un dessin avec une seule commande. Il est aussi possible de couper une sélection avec les touches Ctrl + X pour en garder une copie dans le presse-papier et l'enlever du dessin. Enfin, une sélection peut être supprimée à l'aide de la touche Backspace. Toutes les options du presse-papier se retrouvent au bas de la barre latérale. N'oubliez pas que les options d'annuler et refaire peuvent être très utiles si jamais vous copiez un élément trop de fois ou si vous supprimez une sélection accidentellement.

Utilisation

  • Appuyer sur Ctrl + C pour copier une sélection
  • Appuyer sur Ctrl + V pour coller une sélection
  • Appuyer sur Ctrl + X pour couper une sélection
  • Appuyer sur Ctrl + D pour dupliquer une sélection
  • Appuyer sur Backspace pour supprimer une sélection
  • Appuyer sur les options du presse-papier dans la barre latérale
