Pipette

Description

Dans le cas où vous souhaitez sélectionner une couleur déjà présente sur votre dessin, la pipette vous permet de rapidement la choisir et de l'appliquer pour votre prochain ajout sur la toile. Cet outil est particulièrement utile lorsqu'une couleur présente sur le dessin n'est plus présente dans les dix dernières couleurs sélectionnées. Un clic droit de la souris assigne la couleur principale et un clic gauche assigne la couleur secondaire.

Utilisation

  • Sélectionner l’icône de la pipette dans la barre d’outils
  • Effectuer un clic droit sur une couleur pour la définir comme couleur primaire
  • Effectuer un clic gauche sur une couleur pour la définir comme couleur secondaire
