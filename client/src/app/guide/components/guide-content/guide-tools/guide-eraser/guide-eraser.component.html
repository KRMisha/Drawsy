Efface

Description

Si vous effectuez un heureux petit accident sur votre dessin, ne vous inquietez pas, vous pouvez effacer le tracé fautif avec l'efface. Après avoir selectionné l'outil, vous pourrez vous déplacer au dessus des différents éléments. Vous remarquerez que l'élément choisi pour la suppression est surligné en rouge. Il suffit ensuite d'un seul clic gauche sur l'élément pour l'effacer du dessin. Veuillez prendre note qu'un clic maintenu effacera de façon continuelle les éléments se trouvant sous le curseur.

Utilisation

  • Sélectionner l’icône de l'efface dans la barre d’outils
  • Déplacer la souris sur l'élément à supprimer
  • Effectuer un clic gauche pour effacer l'élément
