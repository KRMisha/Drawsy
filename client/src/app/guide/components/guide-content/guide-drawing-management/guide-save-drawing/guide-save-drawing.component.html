Sauvegarder le dessin

Description

Afin de dévoiler le résultat de votre dessin aux autres utilisateurs de PolyDessin, il vous est possible de sauvegarder votre dessin sur le serveur dans la fenêtre modale de sauvegarde du dessin. Vous avez le choix d'ajouter des étiquettes optionnelles vous qui permettront de retrouver le dessin facilement une fois rendu sur la galerie. Enfin, il suffit d'ajouter un titre au dessin et d'appuyer sur le bouton Sauvegarder le dessin. Veuillez noter que la sauvegarde de modifications d'un dessin chargé depuis la galerie écrasera l'ancienne version du dessin présente sur le serveur.

Utilisation

  • Ajouter des étiquettes au choix
  • Nommer le dessin
  • Sauvegarder le dessin sur le serveur
  • Voir votre dessin affiché dans la gallerie!
